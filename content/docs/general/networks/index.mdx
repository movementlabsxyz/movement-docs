Movement provides several networks with different architectures for development and production needs:

### M1 Networks
1. **[M1 (Mainnet)](/general/networks/mainnet)** — The flagship production network serving real economic value.
2. **[M1 (Testnet)](/general/networks/testnet)** — A native L1 testing environment that provides developers with a safe space to build and test applications using the full Move execution environment before deploying to production networks.


## Timeline and Roadmap

_**Phase 1**_ (Current: Devnet)
focuses on _**core L1 infrastructure development**_ and integrating _**developer tools**_.

_**Phase 2**_ (Testnet Migration)
 involves _**testnet transitions**_ to L1 architecture, creating a _**production-like testing environment**_. Extended testing and validation period with community.

_**Phase 3**_ (Mainnet Migration)
covers _**mainnet transitions**_ from sidechain to L1, achieving _**full decentralization**_, and launching the _**production L1 network**_. A complete native blockchain ecosystem is achieved.
