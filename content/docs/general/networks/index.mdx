Movement provides several networks with different architectures for development and production needs:

### M1 Networks
1. **[M1 (Mainnet)](/general/networks/mainnet)** — The flagship production network serving real economic value.
2. **[M1 (Testnet)](/general/networks/testnet)** — A native L1 testing environment that provides developers with a safe space to build and test applications using the full Move execution environment before deploying to production networks.


## Network Status

Movement is **live on Mainnet**. M1 — the native Move Layer 1 — powers both Mainnet and Testnet, giving developers a full Move execution environment for production applications and for pre-production testing.
