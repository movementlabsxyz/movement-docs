# Run a Validator Node

Running a validator node is not supported on the Movement Network at this time.

## Future System Requirements

When validator nodes become available, they will have specific hardware and software requirements. For detailed specifications and planning information, see the **[System Requirements](/general/nodes/requirements#validator-node---coming-soon)** page.

### Expected Requirements Summary
- **CPU**: 8+ cores minimum (16+ cores recommended for mainnet)
- **Memory**: 32GB RAM minimum (64GB+ recommended)
- **Storage**: 1TB+ NVMe SSD with high IOPS
- **Network**: 1Gbps+ internet connection with low latency
- **Staking**: Minimum 1,000,000 MOVE tokens
